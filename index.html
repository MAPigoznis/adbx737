Bootcamp 2021: Conway's Game of Life

This was one of my first significant programming projects in Java and City Bootcamp has helped significantly in its creation.
The project consists of a grid simulating the well-known Conway's Game of Life on a custom grid as well as some simple UI elements. This was coded using an object oriented approach and several 2D arrays storing grid position data. I also took the opportunity to try out some live data visualisation with the Living Cell graph.

Check out the code on Github: github.com/MAPigoznis
